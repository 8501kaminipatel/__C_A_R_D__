HTML

Hypertext Markup Language (HTML) is a text-based language that's used to create web pages. It's the standard markup language for building web pages.

CSS

Cascading Style Sheets (CSS) is a programming language that controls how HTML and XML documents appear on screen or on paper.

JAVASCRIPT

JavaScript is a programming language that adds interactivity to web pages. It's used to create dynamic web pages that update automatically, like animated graphics and interactive maps.
